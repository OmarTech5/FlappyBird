# FlappyBird
A simple recreation of the famous FlappyBird

# Requirements
      pip install pygame

# Run via Terminal
To make it run open your terminal and run:

      cd '/Users/youruser/Downloads/Flappy-Bird'  #Change this with your real path
      python3 game.py

# Run via VS Code
To be able to see the source code on VS Code simply open the workspace in the "Flappy-Bird" directory
